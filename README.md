## Installation

```bash
$ npm install --global yarn
$ yarn install
```

## Running the app

```bash
# development mode
$ yarn dev
# testing mode
$ yarn test
# production mode
$ yarn start

# ESLint Fix
$ yarn lint

#creating tables in postgresl

$ yarn create_tables
```
